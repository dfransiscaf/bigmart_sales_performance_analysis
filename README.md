# BigMart Sales Performance Analysis


BigMart is a global retail corporation managing over 1,500+ products across various cities. Established in 2011, the company has grown into a data-rich platform, recording 8,500+ transactions and generating over $1.2M in sales revenue.

This project focuses on analyzing 7,000+ transaction records to evaluate store performance and consumer buying patterns. The primary objective was to transform raw data into a dynamic sales dashboard, enabling management to make faster, data-driven decisions regarding product performance and outlet efficiency.

The analysis focuses on the following key areas:
- Sales by Outlet Type & Size: Evaluated revenue contributions across different store formats.
- Product Performance: Identified top-performing item categories to optimize stock levels.
- Historical Performance: Analyzed sales trends based on the outlet's establishment year.
- Geographical Analysis: Compared sales performance across different city tiers (Tier 1, 2, and 3).
